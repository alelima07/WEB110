HTML ELEMENTS

This table provides a brief overview of the basic elements needed to create tables in HTML, including a brief description of the task they perform.

HTML Elements Table
Element Description Block, Inline, N/A Container or Empty
<!--..--> Used for comments in the source code. These comments are not visible on the page. N/A Container
<!DOCTYPE> Declaration; tells browser what type of page it should be. Must be the very first thing in your HTML document, before the <html> tag N/A Empty
<a> Defines a Hyperlink, used for linking one page to another. Includes the href attribute, which indicates the link's destination. Inline Container
<article> Used to semantically specify independent, self-contained content. Used for forum posts, blog posts, news stories, or comments. Block Container
<aside> Used to semantically specify content that is separate from the primary content, but related. Block Container
<body> Tag used to contain all the visual HTML contents Block Container
<br> Line break. Used to insert a single line break. Inline Empty
<dd> Definition in a description list. Used to describe a term or name. Block Container
<div> Tag used to define a division or section in an HTML document, in order to selectively apply css styling elements. Block Container
<dl> The dl tag defines a Description list. Block Container
<dt> Used to define a term or name in a description list. Block Container
<em> Visual and semantic emphasis, appearing as italicized. Compatible with screen readers. Inline Container
<fieldset> Tag used to group related elements in a form. Block Container
<figcaption> Creates a caption for a figure/image element. Block Container
<figure> Semantic way of indicating illustrations, diagrams, photos, code listings, related to the main flow but separate. Block Container
<footer> Defines a footer section of a document. Typically placed at the end of an HTML document to group together information: authorship, copyright, contact, etc. There can be more than one. Block Container
<form> Tag used to create an HTML form for user input. Block Container
<h1> to <h6> Header tags, used to define headings. h1 is the most important, descending in importance to h6. Block Container
<head> Container for all head elements, including title of the document, various styles and scripts, and metatags. Invisble Container
<header> Semantic tag to signify header section. containing elements like heading elements such as logos, icons, and authorship information. Block Container
<html> Container encompassing all other HTML elements. N/A Container
<iframe> A way to embed another HTML document within the current HTML document. Block Empty
<img> Defines an image by referencing a file in the HTML document using the "src" and "alt" attributes. Inline Empty
<input> Form field. Specifies an imput field where a user inputs data into a field. Inline Empty
<label> Form field. Defines a label for an input element in a form. Inline Container
<li> Defines a list item for ordered, unordered, and menu lists. Block Container
<link> Defines a link to external resources such as style sheets or fonts. Only used in the head section. N/A Empty
<meta> Invisible element included in the head section to include data that can be used by browsers such as author, date last modified, etc. N/A Empty
<nav> Tag used to define a section for a major block of navigation elements. Block Container
<ol> Tag used to define an ordered list, which has numerical or alphabetical list identifiers. Block Container
<p> Tag used to define a paragraph; for grouping text. Block Container
<script> Tag that either contains or points to external JavaScript elements. N/A Container
<section> Tag used for defining sections within a document such as headers, footers, etc. Block Container
<strong> Visual and semantic emphasis, appearing as bolded text. Compatible with screen readers. Inline Container
<style> Tag used to define style information for HTML. This should be used to point to CSS stylesheets instead of styling within HTML directly. Inline Container
<table> Tag containing and defining a table, holds all elements related to the table like header, rows, caption, etc. Block Container
<td> Use to define info for a standard cell in a table. Inline Container
<th> Tag used to define a Header cell in a table. Block Container
<title> Used within the head section, the title tag is used to set the title of the page that is seen by browsers. Inline Container
<tr> Tag used to define a Table Row in a table. Block Container
<ul> Tag used for each item in a bulleted list (as opposed to a list that is numerical/alphabetically labelled; this would use the "<ol>" tag). Block Container
